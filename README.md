## Kinopub for tvOS (v2)

*В связи с нехваткой времени этот проект теперь open source :). Присоединяйтесь! Работаем в ветке develop*

### Что требуется для запуска: 

* Платный аккаунт на kino.pub
* Xcode 8 / Swift 3
* Приставка AppleTV 4
* Кабель USB-C

![Screenshot](screenshot.png)
